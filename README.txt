In this project, entities from https://the-internet.herokuapp.com/ are tested by using
Selenium WebDriver and TestNG for test verification.

Entities which were tested are:
-Alerts
-Prompts
-Checkboxes
-Dropdown menus
-Dynamic loading (by immplementing wait strategies)
-Horizontal slider
-Keypresses (with class Keys)
-Hovers

Project is designed with Page Object Model, so, it consists of main and test folders. In main folder, framework and element interactions are
set, and test folder consists only of tests. 

Depending on specific elements, different element locators were used (By: ID, CSS selectors, tagname, xpath..).
